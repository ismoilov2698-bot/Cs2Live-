ANDROID STUDIOSIZ APK CHIQARISH YO'LI

Sizning kompyuteringizda Google Maven / Android plugin yuklanmayapti. Shuning uchun APKni GitHub Actions orqali online build qilamiz.

1) github.com saytiga kiring va login qiling.
2) New repository bosing.
3) Repository name: CS2LiveWallpaperRealmeGT5
4) Public yoki Private tanlang.
5) Create repository bosing.
6) Ushbu ZIP ichidagi HAMMA fayllarni repositoryga upload qiling.
   Muhim: app, build.gradle, settings.gradle va .github papkasi bir joyda bo'lishi kerak.
7) GitHub repository sahifasida Actions bo'limiga kiring.
8) Build APK workflowini tanlang.
9) Run workflow bosing.
10) 3-8 daqiqa kuting.
11) Build tugagach, pastdagi Artifacts bo'limidan CS2LiveWallpaperRealmeGT5-debug-apk faylini yuklab oling.
12) Ichidan app-debug.apk chiqadi.
13) APKni telefoningizga yuboring va o'rnating.
14) Settings > Wallpapers > Live Wallpapers ichidan CS2 Live Wallpaper ni tanlang.
